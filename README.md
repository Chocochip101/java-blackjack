## 기능 목록

### 덱

- [ ] 생성 시점에서 카드를 섞는다.
- [ ] 딜러와 참가자에게 두 장씩 지급한다.

### 플레이어들

- [ ] 딜러는 한 명만 존재한다.
- [ ] 참가자들 사이의 승패 계산한다.
- [ ] 승패를 반환한다.

### 딜러

- [ ] 딜러는 16점 이하면 카드를 추가 지급한다.

### 플레이어

- [ ] 참가자에게 카드를 더 뽑을지 요청한다.

### 카드 손패

- [ ] 들고 있는 카드의 전체 점수를 반환한다.
- [ ] 카드는 점수를 반환한다.
  - [ ] Ace는 1 또는 11점으로 계산한다.
  - [ ] J,Q,K는 10으로 계산한다.
  - [ ] Ace는 21보다 클 경우에는 1로, 그렇지 않으면 11로 계산한다.
- [ ] 점수가 21점이 넘을 경우 카드를 추가로 요청할 수 없다.

### 카드

- [x] 카드의 기호와 숫자를 반환한다.

### 입출력

- [ ] 참가자 이름을 입력 받는다.
    - [ ] 각 참가자는 ,로 구분한다.
- [ ] 게임 완료 후 승패를 출력한다.
- [ ] 카드 분배 결과를 출력한다.
- [ ] 카드를 추가 지급 받을지 물어본다.
    - [ ] 응답은 y/n로만 받을 수 있다.
